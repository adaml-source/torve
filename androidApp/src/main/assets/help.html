Help & Documentation

1. Getting Started

Torve is a media companion app that helps you organize, discover, and track your entertainment across multiple services. Browse and search content, manage your personal library, track what you watch, and tune into your channel playlists — all from a single interface.

To get started, configure your preferred services in Settings, install any content extensions you'd like to use, and optionally connect a tracking account to sync your watch history.

2. Cloud Services

Torve supports connecting to cloud media services that provide high-speed access to your personal content library.

Go to Settings > Cloud Service to connect your preferred provider. You can authenticate via:

  • API Key — paste your key from the provider's account page
  • Device Authorization — scan a code on the provider's website to grant access securely

You can link multiple providers simultaneously. Torve will check availability across all connected services and prioritize the best results.

3. Content Extensions

Torve uses a modular extension system to discover and resolve media links from your connected services.

Go to Settings > Extensions to manage your installed extensions. To install one:

  • Paste the extension's manifest URL (usually ends in /manifest.json)
  • The extension will be validated and added to your list

Extensions provide catalog browsing and link resolution from your connected cloud services.

4. Channels

Navigate to the Channels tab to manage your personal channel playlists.

Playlists

Import your personal playlist by pasting its URL. The app parses channel names, logos, group titles, and program guide data.

Provider Login

If your playlist provider supports direct login, enter the server URL, username, and password to import your subscribed content.

You can manage multiple playlists, filter by category, mark favorites, and view program guides when available.

5. Trakt Integration

Trakt.tv provides comprehensive watch tracking, ratings, and social features for movies and TV shows.

Go to Settings > Integrations > Trakt to connect:

  • Tap Connect to start device authorization
  • Visit the URL shown and enter the code
  • Once authorized, your watchlist and history will sync

Features

  • Scrobbling — automatically marks content as watched when you finish
  • Two-way sync — changes in Trakt appear in the app, and vice versa
  • Watchlist — your Trakt watchlist appears on the home screen
  • Ratings — rate content and sync ratings with Trakt

Use the Sync now button to manually trigger a full sync at any time.

6. SIMKL Integration

SIMKL is focused on anime and TV show tracking with automatic detection.

Go to Settings > Integrations > SIMKL and authorize via device code. Your watch history will sync with SIMKL automatically.

7. MDBList Integration

MDBList provides aggregated ratings from multiple sources and curated content lists.

Go to Settings > MDBList:

  • Enter your free API key from mdblist.com/preferences
  • Search for public lists and add them to your home screen
  • Ratings from IMDb, Rotten Tomatoes, TMDB, Metacritic, Letterboxd, Trakt, MAL, and more will appear on content cards

8. AI-Powered Search

Torve includes AI-powered search that understands natural language queries like "sci-fi movies from the 90s with time travel" or "feel-good comedies similar to The Grand Budapest Hotel."

Go to Settings > AI Provider to configure. Supported providers:

  • Claude (Anthropic)
  • ChatGPT (OpenAI)
  • Gemini (Google)
  • Perplexity
  • DeepSeek

Enter your API key for your preferred provider. AI search results appear alongside standard search results to help you discover content that matches your mood and preferences.

9. Rating Pills

Rating pills show scores from multiple providers directly on poster and landscape cards.

Go to Settings > Appearance > Rating Pills to configure:

  • Pill style — choose between "With Icon" (provider logo) or "With Letter" (letter abbreviation)
  • Position — inside the card image or below it
  • Max ratings — limit how many pills appear per card (1–9)
  • Provider order — drag to reorder which ratings appear first
  • Per-source toggle — enable or disable individual rating sources
TMDB ratings appear automatically. For IMDb, Rotten Tomatoes, and Metacritic, add a free OMDB API key in Settings > Integrations > OMDB. For additional sources like Letterboxd and MAL, add an MDBList API key.

10. Card Styles & Home Layout

Card style presets control how content cards look across different sections of the home screen.

Go to Settings > Appearance > Card Styles to create and manage presets. Each preset includes size, hover effects, watched indicators, and rating pill configuration.

In Settings > Home Layout, you can:

  • Drag to reorder home sections
  • Enable or disable sections
  • Assign a specific card style preset to each section
  • Set custom titles for sections

Sections use the built-in default style unless you explicitly assign a different preset.

11. Regex Filters

Regex patterns let you filter out unwanted results by title matching.

Go to Settings > Regex Patterns:

  • Use Quick Add presets for common filters (quality preferences, language filters, etc.)
  • Create custom patterns using standard regex syntax
  • Toggle individual patterns on/off without deleting them
Use (?i) at the start for case-insensitive matching. Use | to match alternatives. Example: (?i)(cam|ts) excludes results containing "cam" or "ts".

12. Kodi Remote

Send media directly to your Kodi devices for playback on your TV.

Go to Settings > Kodi Remote:

  • Add devices with name, IP address, and port (default 8080)
  • Use the Test button to verify connectivity
  • When viewing media details, choose "Play on Kodi" to send it to a connected device

13. External Players

When playing content, you can choose to open it in an external video player:

  • VLC — broad codec support
  • MX Player — hardware acceleration
  • Any other compatible video player installed on your device

14. Jellyfin & Plex

Connect your personal Jellyfin or Plex media server to overlay your own library on the home screen.

Go to Settings > Integrations:

  • Jellyfin — enter your server URL and API key, then select a user profile
  • Plex — enter your server URL and access token

Items from your personal library will be highlighted, and "Continue Watching" from your media servers appears alongside other content on the home screen.

15. Quality & Playback Preferences

Go to Settings > Playback Preferences to control content selection:

  • Resolution range — set minimum and maximum quality (SD to 4K)
  • File size limit — cap the maximum file size for downloads or playback
  • Availability filter — show only readily available content from your connected services
  • Codec preference — prefer HEVC, AV1, or no preference
  • HDR mode — auto, force on, or force off

16. Backup & Restore

Go to Settings > Backup & Sync:

  • Export — saves your extensions, preferences, watch progress, and favorites to a file
  • Import — restores from a previously exported file
API keys and authentication tokens are never included in exports for security.

17. Diagnostics

Go to Settings > Diagnostics to view a summary of your app configuration:

  • Service connection status for all configured integrations
  • Sync timestamps
  • Device and app version information
  • Extension, playlist, download, and card style details

Use the Share or Copy buttons to send diagnostics when requesting support.

18. Navigation & Tips

  • Use the bottom tabs to switch between Movies, TV Shows, Channels, and Settings
  • Pull down on the home screen to refresh content
  • Long-press on a content card for quick actions
  • The app supports Android TV with D-pad navigation and leanback UI

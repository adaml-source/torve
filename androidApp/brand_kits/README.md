# Brand Kits (Provider Logos)

This folder holds provider logo sources and (optionally) downloaded official brand assets.

## Status
- Official brand pages are listed per provider in `SOURCES.md`.
- Assets are NOT downloaded unless they are publicly available and usage terms allow it.
- If a provider requires login/approval, note it in the provider's `SOURCES.md`.

## Structure
- `androidApp/brand_kits/<Provider>/SOURCES.md`
- (Optional) `androidApp/brand_kits/<Provider>/logo.svg`
- (Optional) `androidApp/brand_kits/<Provider>/logo.png`

## Next
When you have an official source for a provider, add it to that provider's `SOURCES.md`.
